    • About Me

    Building web applications that are easy to use and reliable are of my highest priorities.

    I believe that software should work for users.
    Users should NOT have to work for the software.

    I prefer working with teams that keep everything planned.

    I've learned that having a solid design before coding enhances the project's success.

    I've learned that working thru all combinations of test scenarios (when possible) produces the most reliable code.

    I'm most familiar with using Node.js, Express.js, Vue.js and MySQL technologies.

    Pet Purfect

    This full stack web application was designed and built by a team of 4 web developers. The app uses a React.js front-end and a mySQL database on the back-end. Sequelize.js is used as the object relational mapper to the database. Firebase Storage is used to hold uploaded images. The server uses Node.js and Express.js libraries. This app uses a model-view-controller (MVC) folder structure. And, Passport.js is used for the user authentication.
